    Ataque SSH.¶

    Escaneo del servicio.¶

    Escaneamos a ver si el equipo tiene SSH activado.

    • -sV: Hace un sondeo de versiones que nos permite diferenciar entre los puertos abiertos de los filtrados. Además devuelve información del equipo y versión donde se ejecuta el servicio SSH.
    • -n: No intenta hacer resolución de nombres. Simplemente para ganar tiempo.

    Tipo de Cifrados Soportados.¶

    • Algoritmos Diffie-Hellman (kex-algorithms).
    • Cifrado Asimétricos (server_host_key_algorithms).
    • Cifrados Simétricos (encryption_algorithms).
    • Cifrados HMAC (mac_algorithms).

    Creación y uso de wordlists.¶

    Vamos a probar un ataque por diccionario, para ello necesitamos un diccionario o una lista con todas las posibilidades a probar.

    Podemos hacer uso del comando crunch para crear nuestra propia lista de palabras.

    Habiendo hecho un poco de ingeniería social, por ejemplo averiguando el nombre de usuario, podemos crear una lista más específica.

    crunch 9 9 -t salva%%%% -l @@@@aaaa

    Otra opción es simplemente usar una wordlist prefabricada por otras personas con las contraseñas más habituales. Además dentro de Kali ya vienen integradas algunas.

    #En este directorio se encuentran las wordlists por defecto de Kali.
ls /ush/share/wordlists
#En caso de querer descomprimir alguana.
gzip -d  /usr/share/wordlists/rockyou.txt.gz

    Herramientas para lanzar el ataque.¶

    Hydra.¶

    • -l: Especificamos el usuario, si lo desconocemos podemos hacer uso de usuarios habituales como “admin”, “root”, etc.
    • -P: Especifica la lista de palabras (wordlist) a usar.
    • Finalmente indicamos la IP de la víctima y el protocolo a usar.

    Medusa.¶

    • -T 5: Nos indica el tiempo de espera entre intentos. El modo 0 es un modo lento para evitar equipos de seguridad que pudieran detectar muchos intentos de acceso. El modo 5 es el más rápido, no espera entre intentos.
    • Como en el caso anterior hay que especificar el usuario, wordlist, IP de la víctima y el puerto/protocolo.
    medusa -h 10.0.0.1 -u lab -P /usr/share/wordlists/rockyou.txt -M ssh

    Módulos

    Es importante saber que cada herramienta tiene módulos para diferentes protocolos.

    ncrack

    Existen otras herramientas como ncrack.

    Solución.¶

    • Configurar en SSH el máximo número de intentos de registro.
    • Configurar una contraseña larga con mayúsculas, minúsculas, signos, números y que no contenga palabras habituales/nombres.
    • Usar certificados digitales.
